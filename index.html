Features

  • Photorealistic 3D coin rendering (OpenGL, Pygame)
  • Chrome/metallic edge and rim effects
  • Animated circuitry and pulse effects
  • Endless coin rotation
  • Signature and watermark overlays

How to Run

  1. Install Python 3.10+ and dependencies: pip install pygame PyOpenGL Pillow
  2. Run: python src/main.py

WebGL Version

See index_webgl.html for a browser/mobile-friendly version (Three.js, placeholder only).
