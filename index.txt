# Digital Coin App Project Index

## Main Files
- src/main.py: Main application logic and rendering
- darkcoin4.png: Coin front face image
- ohcrlogoGREENBACK.png: Coin back face image
- gbaileysignature.png: Signature overlay image
- gbailey_watermark.png: Watermark overlay image

## Features
- Photorealistic 3D coin rendering (OpenGL, Pygame)
- Chrome/metallic edge and rim effects
- Animated circuitry and pulse effects
- Endless coin rotation
- Signature and watermark overlays

## Usage
Run with:
```
python src/main.py
```
